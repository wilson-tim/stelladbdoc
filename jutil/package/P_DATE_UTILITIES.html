/* ***************************************************************************** *

Converts a date/time in one time zone to another time zone.

Notes:

1. A DATE datatype is used for the date/time to be converted not a datatype with the
   times zone already included.
   
2. This function tries to use the ORACLE time zone features first but if this fails it
   then tries to use a Java class if the date is at GMT. 
   
3. If the date/time cannot be converted then a NULL value is returned. 

%param p_in_datetime Date/time to be converted to another time zone.
%param p_in_datetime_timezone Timezone of date/time to be converted.
