  Purpose
  -------
  Wrapper function for Java class that returns offset (in milliseconds) of date/time
  in time zone to GMT. It is assumed that the date/time being passed is in GMT.

  Date      By          Description                                         Version

  Name             Description                                         Default In/Out

  p_in_tz_name     Time zone to get offset for.                                  IN
  p_in_era         Era date/time relates to ('BC' or 'AD').                      IN
  p_in_year        Year number of date/time.                                     IN
  p_in_month_no    Month number of date/time.                                    IN
  p_in_day_no      Day number of date/time.                                      IN
  p_in_day_name    Name of day (e.g. 'WEDNESDAY') of date/time.                  IN
  p_in_day_time_   Time of dat/time in milliseconds.                             IN
